The folder recommender: 4 python files:
	+ _decider.py			: Include the combination of SH Recommender and probability score
	+ predictor.py			: Read the probability score from database
	+ _recommender.py		: Deploy the output of the new recommenders and SH recommender which the other team require for
	+ run_recommender_manually.py	: Test all the deciders

The folder notebook:
	+ probability_score_generator.ipynb	: Extract features, train Logistic Regression model and generate the probability score which is saved in the database
	+ analyze_result.ipynb			: Analyze the results 
	+ survey_result.csv 			: Data of the third research question: Users' satisfaction level.

survey.pdf : The questionnaire asking for user satisfaction level for question 3.
